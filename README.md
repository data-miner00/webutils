# webutils

Useful web utilities for developers in one place.

## Getting Started

Install dependencies using NPM.

```
npm i
```

Starting development server.

```
npm start
```

Build for production.

```
npm run build
```

Building Docker image.

```
docker build . -t webutils
```

Or use docker-compose.

```
docker-compose up -d
```
